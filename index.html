O que é o Hackathon?

Inovação

Uma maratona de programação que desafia equipes a desenvolverem soluções tecnológicas inovadoras para o dia a dia.

Colaboração

Um ambiente colaborativo para troca de experiências e ideias entre os próprios alunos.

Transformação / Impacto

Desenvolvimento de projetos que podem facilitar tarefas e melhorar a rotina das pessoas.

Desafios

Calculadora Personalizada

Uma calculadora que vai além de operações básicas, permitindo ao usuário realizar cálculos específicos do dia a dia.

  • Interface com botões numéricos e operações básicas
  • Modo personalizado para cálculos específicos
  • Funções para calcular gorjetas
  • Divisão de contas entre amigos
  • Conversão de moedas

Cronômetro/Temporizador

Uma ferramenta para medir tempo útil para estudar, cozinhar ou fazer exercícios.

  • Botões para iniciar, pausar e zerar
  • Exibição do tempo em formato claro
  • Alarme sonoro ou visual
  • Modo Pomodoro com ciclos pré-definidos
  • Interface intuitiva e responsiva

Gerador de Ideias

Uma aplicação que sugere ideias aleatórias para inspirar o usuário.

  • Categorias selecionáveis
  • Botão "Gerar" para exibir sugestões
  • Banco de dados com 10+ ideias
  • Design temático e atraente
  • Interface intuitiva

Lista de Compras

Um sistema para organizar itens a comprar, com foco em praticidade.

  • Adição e remoção de itens
  • Marcação de itens comprados
  • Quantidade por item
  • Soma total estimada
  • Interface responsiva e intuitiva

Planejador de Rotina

Uma ferramenta para organizar o dia do usuário.

  • Adição de tarefas com horário
  • Lista visível das tarefas do dia
  • Marcação de tarefas concluídas
  • Interface organizada e clara
  • Design responsivo
